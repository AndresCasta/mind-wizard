# BashScripts
## push.cmd
Util para hacer push a develop en un solo comando.

```bash
push "This is the commit description"
```

## start_server.cmd
A primera vista inofencivo, pero va a ser de gran utilidad cuando necesites iniciar varios servidores a la vez.

Asegurate de poner el nombre de tu juego dentro del script.

*NOTA: El operador **=** en bash es sensible a espacios.*:
```bash
set gameName=YourGameNameHere
```
